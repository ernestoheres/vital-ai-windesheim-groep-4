# Data Science Assignment – Sepsis Prediction (Isala x Windesheim)
This repository supports a student assignment developed in collaboration with Isala Hospital, HBO-ICT, and the research group IT-innovations in Healthcare. The assignment focuses on the early detection of sepsis using data science and machine learning techniques.

## Context
Sepsis is a life-threatening condition that requires early detection and intervention. In clinical practice, timely recognition remains challenging, offering opportunities for data-driven solutions.

## Assignment
Students will work with (public) datasets, including data from the [PhysioNet Challenge 2019](https://physionet.org/content/challenge-2019/1.0.0/), to:

- Develop models for early prediction of sepsis

and, one of the following sub-goals:
- Explore fairness (e.g., differences across age and gender)
- Reflect on applicability in a real hospital setting
- Consider explainability and transparency of models

## Repository contents
- `/data` – datasets or links to datasets
- `/docs` – assignment description and supporting materials

## Goal
The goal is not only to build a technically sound model, but also to understand how data science solutions can create value in healthcare practice. Focus on making informed design choices rather than optimizing performance alone.
